# Customer Behavior Analysis & Business Intelligence Dashboard

## Overview

This project focuses on analyzing customer purchasing behavior to uncover meaningful business insights and support data-driven decision-making. Using Python, SQL, Excel, and Power BI, the project performs data cleaning, exploratory data analysis (EDA), customer segmentation, and dashboard development to identify purchasing trends, revenue patterns, and customer preferences.

## Objectives

* Analyze customer purchasing behavior and spending patterns.
* Identify high-value customers and customer segments.
* Evaluate product and category performance.
* Generate actionable business insights through data visualization.
* Build an interactive Power BI dashboard for decision-making.

## Tech Stack

* Python

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn
* SQL
* Microsoft Excel
* Power BI
* Jupyter Notebook

## Project Workflow

### 1. Data Collection

* Imported customer transaction data.
* Validated data quality and structure.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected inconsistent records.
* Standardized data formats.

### 3. Exploratory Data Analysis (EDA)

* Customer purchasing trends.
* Revenue analysis.
* Product performance evaluation.
* Customer segmentation.
* Sales distribution analysis.

### 4. Data Visualization

Created visualizations to analyze:

* Revenue trends
* Customer segments
* Product category performance
* Purchase frequency
* Sales contribution by customer groups

### 5. Dashboard Development

Built an interactive Power BI dashboard featuring:

* Total Revenue
* Average Order Value (AOV)
* Customer Segmentation
* Product Performance
* Sales Trends
* Key Business KPIs

## Key Insights

* Identified customer groups contributing the highest revenue.
* Analyzed purchasing behavior across different segments.
* Detected top-performing products and categories.
* Revealed patterns useful for customer retention strategies.
* Generated business recommendations based on analytical findings.

## Results

The project successfully transformed raw customer transaction data into meaningful insights through analytics and visualization. The interactive dashboard enables stakeholders to monitor performance, understand customer behavior, and make informed business decisions.
